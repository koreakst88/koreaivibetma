## Шаг 1 — Установи редактор

Чтобы работать с файлами приложения, нам нужна одна программа.  
Она называется **VS Code**.

Это просто удобная программа для работы с файлами. Как Word — только для кода.  
Но не пугайся слова "код" — ты не будешь ничего писать руками 😊

### Как установить:

1. Открой браузер на компьютере
2. Перейди на сайт: **code.visualstudio.com**
3. Нажми большую кнопку **Download**
4. Установи программу как обычно (Next → Next → Finish)

Всё. VS Code установлен ✅

---

## Шаг 2 — Подготовь папку

Теперь создадим место, где будет жить твоё приложение.

1. Создай новую папку на рабочем столе
2. Назови её: **my-first-app**
3. Открой VS Code
4. Нажми **File → Open Folder** и выбери папку **my-first-app**

Теперь VS Code смотрит в твою папку. Отлично ✅

---

## Шаг 3 — Твой первый промпт

Вот здесь начинается самое интересное.

Ты не будешь писать код сам.  
Вместо тебя это сделает **искусственный интеллект** — по твоей команде.

Такая команда называется **промпт**. Это просто текст, который ты отправляешь нейронке.  
Нейронка читает — и делает всё сама.

### Выбери нейронку

Можно использовать любую из трёх — все отлично справятся:

| Нейронка | Сайт |
|----------|------|
| 🤖 Claude | claude.ai |
| 🤖 ChatGPT | chat.openai.com |
| 🤖 Gemini | gemini.google.com |

Зайди на любой из этих сайтов. Если нет аккаунта — зарегистрируйся, это бесплатно.

---

### Скопируй этот промпт и отправь нейронке:

> Создай красивый HTML-файл — экран мобильного приложения с каталогом пиццы.
>
> Требования:
> - Дизайн современный, тёмный фон, яркие акценты
> - Вверху заголовок "🍕 Меню пиццы"
> - 4 карточки пиццы: название, короткое описание, цена, кнопка "Добавить"
> - Карточки красивые, с закруглёнными углами и тенями
> - Всё в одном файле index.html
> - Без внешних библиотек — только HTML и CSS внутри файла
> - Адаптировано под мобильный экран

Просто скопируй этот текст, вставь в чат с нейронкой и нажми отправить.

---

## Шаг 4 — Сохрани результат

Нейронка пришлёт тебе готовый код.

Он будет выглядеть примерно так — большой текст, начинается с `<!DOCTYPE html>`.  
Не пугайся — это просто файл твоего приложения.

### Что делать дальше:

1. Выдели весь текст который прислала нейронка
2. Скопируй его (Ctrl+C на Windows / Cmd+C на Mac)
3. В VS Code нажми **File → New File**
4. Вставь скопированный текст (Ctrl+V / Cmd+V)
5. Сохрани файл: **File → Save As**
6. Назови файл: **index.html**
7. Сохрани его в папку **my-first-app**

Готово ✅

---

## Шаг 5 — Открой своё приложение

Теперь самый приятный момент 🎉

1. Открой папку **my-first-app** на компьютере
2. Найди файл **index.html**
3. Дважды кликни на него
4. Он откроется в браузере

Ты увидишь свой первый экран приложения.
