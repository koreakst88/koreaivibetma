## Шаг 1: Обзор Telegram WebApp SDK + Подключение и инициализация (25 мин)

### Часть 1: Что такое Telegram WebApp SDK?

**SDK** — это Software Development Kit (набор инструментов для разработки).

**Telegram WebApp SDK** — это JavaScript-библиотека от Telegram. Она даёт вашему приложению суперспособности внутри Telegram.

| Без SDK | С SDK |
|---|---|
| Ваше приложение — просто сайт, открытый в Telegram | Приложение «знает», что оно внутри Telegram |
| Не знает, кто пользователь | Получает данные пользователя (имя, ID, username) |
| Не имеет доступа к функциям Telegram | Может использовать кнопки Telegram (MainButton, BackButton) |
| Выглядит как обычная веб-страница | Адаптируется под тему Telegram (тёмная/светлая) |
| | Может вибрировать (Haptic Feedback) |
| | Показывает нативные окна Telegram (Alert, Confirm, Popup) |
| | Может закрывать себя, открывать ссылки, запрашивать разрешения |

Это превращает ваш сайт в настоящее Telegram-приложение!

---

### Часть 2: Основные возможности SDK

Давайте посмотрим, что именно можно делать с SDK.

#### 1. Получение данных пользователя

```javascript
const user = Telegram.WebApp.initDataUnsafe.user;
console.log(user.first_name); // "Иван"
console.log(user.id); // 123456789
console.log(user.username); // "@ivan"
```

**Зачем это нужно?**
- Автоматически подставить имя в форму заказа
- Идентифицировать пользователя (сохранить его заказы)
- Персонализировать приветствие: «Привет, Иван!»

---

#### 2. MainButton (главная кнопка)

Это большая кнопка внизу Telegram, как у обычных ботов.

```javascript
Telegram.WebApp.MainButton.text = "Оформить заказ • 1250₽";
Telegram.WebApp.MainButton.show();
```

**Зачем?**
- Удобнее, чем HTML-кнопка
- Всегда на виду (фиксирована внизу)
- Выглядит нативно для Telegram

---

#### 3. BackButton (кнопка «Назад»)

Появляется слева вверху (где обычно стрелка назад в Telegram).

```javascript
Telegram.WebApp.BackButton.show();
```

**Зачем?**
- Привычная навигация для пользователей Telegram
- Не нужна HTML-кнопка «← Назад»

---

#### 4. Haptic Feedback (вибрация)

```javascript
Telegram.WebApp.HapticFeedback.impactOccurred('light'); // Лёгкая вибрация
Telegram.WebApp.HapticFeedback.notificationOccurred('success'); // Вибрация успеха
```

**Зачем?**
- Тактильная обратная связь (как в нативных приложениях)
- При добавлении в корзину → вибрация
- При успешном заказе → вибрация

---

#### 5. Theme (тема оформления)

```javascript
const theme = Telegram.WebApp.themeParams;
document.body.style.backgroundColor = theme.bg_color; // Фон как в Telegram
```

**Зачем?**
- Приложение выглядит как часть Telegram
- В тёмной теме — тёмные цвета
- В светлой теме — светлые цвета

---

#### 6. Popups и Alerts

```javascript
Telegram.WebApp.showAlert("Заказ оформлен!"); // Нативное окно Telegram
Telegram.WebApp.showConfirm("Очистить корзину?"); // Подтверждение
```

**Зачем?**
- Выглядят как окна Telegram (не браузерные alert)
- Более красивые и нативные

---

#### 7. Закрытие приложения

```javascript
Telegram.WebApp.close(); // Закрыть Mini App
```

**Зачем?**
- После оформления заказа → автоматически закрыть приложение
- Или кнопка «Выход»

## Шаг 2: MainButton + BackButton (25 мин)

### Часть 1: MainButton — главная кнопка

#### Что это и зачем?

**MainButton** — это большая кнопка внизу экрана Telegram. Вы её видели в ботах:

- При выборе товара → «Добавить в корзину»
- При оформлении → «Оплатить»
- При отправке формы → «Отправить»

**Преимущества перед HTML-кнопкой:**
- ✅ Всегда на виду — фиксирована внизу, не прокручивается
- ✅ Нативный вид — выглядит как часть Telegram
- ✅ Адаптируется под тему — в тёмной теме тёмная, в светлой светлая
- ✅ Крупная и удобная — легко нажать на телефоне
- ✅ Показывает прогресс — можно добавить loader

Мы заменим HTML-кнопку «Оформить заказ» на MainButton.

#### Где использовать MainButton в нашем приложении?

В нашем приложении MainButton будет:

**1. На странице корзины (cart.html):**
- Текст: «Оформить заказ • 1250₽» (сумма динамическая)

- Скрыта (не показывается)

### Часть 2: BackButton — кнопка «Назад»

## Шаг 3: Адаптация под темы Telegram + UX-улучшения (25 мин)

### Часть 1: Адаптация под темы Telegram

#### Что такое темы Telegram?

Telegram поддерживает разные темы оформления:
- Светлая тема (Light Mode)
- Тёмная тема (Dark Mode)
- Пользовательские темы (разные цвета)

Когда пользователь меняет тему в настройках Telegram, вся цветовая схема меняется.

**Проблема без адаптации:**

Ваше приложение имеет фиксированные цвета:
- Белый фон
- Чёрный текст
- Красные кнопки

Если пользователь включит тёмную тему:
- Telegram станет тёмным
- Но ваше приложение останется белым
- Это выглядит неестественно — как будто приложение не часть Telegram

**Решение:**

SDK предоставляет цвета текущей темы через `Telegram.WebApp.themeParams`. Мы адаптируем наше приложение, чтобы оно автоматически подстраивалось под тему пользователя.

---

#### Какие цвета предоставляет SDK?

SDK даёт следующие цвета:

```javascript
const theme = Telegram.WebApp.themeParams;

// Основные цвета
theme.bg_color          // Цвет фона (#ffffff в светлой, #1e1e1e в тёмной)
theme.text_color        // Цвет текста (#000000 в светлой, #ffffff в тёмной)
theme.hint_color        // Цвет подсказок (серый)
theme.link_color        // Цвет ссылок (синий)
theme.button_color      // Цвет кнопок
theme.button_text_color // Цвет текста на кнопках

// Дополнительные цвета
theme.secondary_bg_color    // Вторичный фон (для карточек)
theme.header_bg_color       // Цвет заголовка
theme.accent_text_color     // Акцентный цвет текста
theme.destructive_text_color // Цвет для удаления (красный)
```

Мы возьмём эти цвета и применим к нашему приложению. Так оно будет выглядеть гармонично в любой теме!
