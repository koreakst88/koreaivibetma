### 4. Логика корзины

**День:** 3, Шаг 2  
**Назначение:** Добавление работающей корзины с сохранением в LocalStorage

```
Добавь функционал корзины в приложение доставки пиццы.

ЗАДАЧА 1: Логика добавления в корзину

В файле src/script.js:

1. Создай массив cart = [] для хранения товаров
2. При клике на кнопку "Добавить в корзину":
   - Получить данные пиццы (название, цена, id)
   - Проверить, есть ли уже эта пицца в корзине
   - Если есть → увеличить количество на 1
   - Если нет → добавить новый товар с количеством = 1
   - Сохранить корзину в LocalStorage: localStorage.setItem('cart', JSON.stringify(cart))
   - Обновить счётчик корзины на кнопке

3. Функция обновления счётчика:
   - Посчитать общее количество товаров в корзине
   - Отобразить число на кнопке "Корзина (число)"

4. При загрузке страницы:
   - Прочитать корзину из LocalStorage: JSON.parse(localStorage.getItem('cart'))
   - Если есть сохранённые товары → загрузить их в массив cart
   - Обновить счётчик

ЗАДАЧА 2: Изменить alert на уведомление

Вместо alert("Пицца добавлена!") показать красивое всплывающее уведомление:
- Текст: "✓ Добавлено в корзину"
- Появляется вверху экрана
- Исчезает через 2 секунды
- Зелёный фон

ЗАДАЧА 3: Обновить HTML

В файле src/index.html:
- У каждой кнопки "Добавить в корзину" добавить атрибут data-id с уникальным ID пиццы (1, 2, 3, 4, 5, 6)
- У кнопки "Корзина" добавить span для счётчика: <span id="cart-count">0</span>

Используй vanilla JavaScript, без фреймворков.
Комментарии на русском языке.
```

**Примечания:** ✅ Работает. LocalStorage сохраняет данные после перезагрузки.

```
Создай страницу корзины в файле src/cart.html.

Заголовок:
- "Корзина 🛒"

Список товаров:
- Если корзина пуста: показать "Корзина пуста. Добавьте пиццы!"
- Если есть товары: отобразить список

Каждый товар в списке:
- Название пиццы
- Цена за штуку
- Количество (с кнопками "+" и "−")
- Сумма за этот товар (цена × количество)
- Кнопка "Удалить" (красная)

Внизу страницы:
- Итоговая сумма: "Итого: ХХХ ₽" (крупным шрифтом)

   - Обновить LocalStorage
   - Обновить отображение

5. Кнопка "Очистить корзину":
   - Очистить массив cart
   - Удалить из LocalStorage
   - Показать "Корзина пуста"

6. Кнопка "Оформить заказ":
   - Перейти на страницу order.html (создадим позже)

СТИЛИ (src/style.css):

- Адаптивный дизайн
- Карточки товаров с тенью
- Кнопки +/− в виде круглых иконок
- Красивая анимация при удалении товара

Подключи cart.html к существующим style.css и script.js.
Используй ту же цветовую схему (красный #E63946 + белый).
```

## Советы по использованию промптов

### 1. Адаптация под свой проект

```
❌ Не копируйте слепо:
"Добавь пиццы..."

✅ Адаптируйте:
"Добавь [ваши товары]..."
```

### 2. Уточнение деталей

Если результат не идеален с первого раза:

```
"Хорошо, но сделай кнопку больше"
"Измени цвет на #007BFF"
"Добавь анимацию при наведении"
```

### 3. Комбинирование промптов

Можно объединять несколько промптов:

```
"Выполни задачи 4 (Логика корзины) и 5 (Страница корзины) вместе"
```

### 4. Сохранение своих промптов

Когда создаёте что-то новое — сохраняйте в `prompts.md`:

```markdown
## Мой промпт для [функция]
[текст промпта]
```
