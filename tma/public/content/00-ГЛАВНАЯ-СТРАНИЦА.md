## 🎓 СТРУКТУРА КУРСА

### День 0: Подготовка и инструменты (90 мин)
**Цель:** Установить все необходимое и создать первый проект

**Темы:**
- Презентация курса и знакомство
- Обзор инструментов (ChatGPT, Claude)
- Установка: VS Code, Node.js, Git, GitHub, Vercel
- Основы терминала
- Первая веб-страница через ИИ
- Создание Telegram бота

**Результат:** Готовая среда разработки + первый бот

---

### День 1: Принципы Vibe Coding (60 мин)
**Цель:** Научиться управлять ИИ как инструментом

**Темы:**
- Философия Vibe Coding
- 4 роли человека (Vision, Architecture, QC, Iteration)
- Формула эффективных промптов
- Разбивка проектов на блоки
- Типичные ошибки новичков
- Первая интерактивная функция

**Результат:** Карточка пиццы с рабочей кнопкой

---

### День 2: Архитектура Telegram Mini Apps (60 мин)
**Цель:** Создать и задеплоить первое Mini App

**Результат:** Работающее приложение в Telegram

### День 3: Корзина и хранение данных (60 мин)
**Цель:** Добавить корзину с сохранением данных

- Создание корзины через Cline
- Улучшение UI/UX корзины
- Форма оформления заказа с валидацией
- Экран подтверждения

**Результат:** Полный цикл заказа работает

## 🛠️ ТЕХНОЛОГИЧЕСКИЙ СТЕК

**Инструменты разработки:**
- VS Code (редактор кода)
- Cline (расширение для VS Code, AI-ассистент)
- ChatGPT / Claude (AI для генерации кода)
- Terminal (командная строка)

**Технологии:**
- HTML, CSS, JavaScript (vanilla)
- Telegram WebApp SDK
- Google Apps Script
- LocalStorage

**Сервисы:**
- GitHub (хранение кода)
- Vercel (хостинг)
- Google Sheets (база данных)
- Make (автоматизация)
- Google Analytics (аналитика)

**Не требуется знание:**
- ❌ Программирования
- ❌ Фреймворков (React, Vue и т.д.)
- ❌ Backend разработки
- ❌ Баз данных
