## ОБЗОР ПРОЕКТА

### Назначение

Telegram Mini App для доставки образовательных материалов курса "Как создавать цифровые проекты с помощью ИИ".

### Ключевые особенности

- 📚 Материалы курса по 7 дням обучения
- 🔒 Постепенное открытие доступа через коды разблокировки
- 📖 Библиотека из 14+ готовых промптов
- 📋 Шпаргалки и чек-листы
- ✅ Интерактивные чек-листы с сохранением прогресса
- 💬 Прямая связь с преподавателем
- 📱 Адаптация под тему Telegram (светлая/тёмная)

### Целевая аудитория

Ученики курса — новички без опыта программирования, изучающие Vibe Coding.

### Формат использования

- **Преподаватель:** Проводит индивидуальную встречу → даёт код доступа
- **Ученик:** Вводит код в боте → открывается новый день → изучает материалы в TMA

```
┌─────────────────────────────────────────────┐

│  └─ TMA (Vercel)                            │
│      ├─ Главная (список дней)              │
│      ├─ Страница дня (материалы)           │
│      ├─ Библиотека промптов                │
│      └─ Шпаргалки                           │
│                                             │
└─────────────────────────────────────────────┘
              ↓
┌─────────────────────────────────────────────┐
│  ПРЕПОДАВАТЕЛЬ                              │
│  ├─ Проводит встречу                        │
│  ├─ Даёт код разблокировки (DAY1, DAY2...)  │
│  └─ Отвечает на вопросы в чате              │
└─────────────────────────────────────────────┘
```

### Поток данных
```
1. Ученик открывает бота
   ↓
2. Бот показывает кнопку "Открыть курс"
   ↓
3. TMA загружается из Vercel
   ↓
4. TMA проверяет разблокированные дни (LocalStorage)
   ↓
5. Показывает доступные дни
   ↓
6. Ученик вводит код в боте
   ↓
7. Бот отправляет callback в TMA (через URL параметр)
   ↓
8. TMA сохраняет разблокировку в LocalStorage
   ↓
9. День открывается
```

### Компоненты системы

**1. Telegram Bot (Node.js + Grammy)**
- Приветствие и навигация
- Приём и валидация кодов разблокировки
- Передача unlock статуса в TMA

**2. TMA Frontend (HTML/CSS/JS + Tailwind)**
- Отображение материалов
- Навигация по дням
- Контроль доступа (проверка LocalStorage)
- Рендеринг Markdown → HTML

**3. Content (Markdown файлы)**
- 7 дней курса (day-0.md ... day-7.md)
- Библиотека промптов (prompts.json)
- Шпаргалки (cheatsheets.json)

**4. LocalStorage (клиентское хранилище)**
- Разблокированные дни
- Прогресс по чек-листам
- Настройки (опционально)

```javascript
// js/day.js (фрагмент)

async function loadDayContent() {
    const urlParams = new URLSearchParams(window.location.search);
    const dayId = urlParams.get('id'); // например: "day1"
    
    // Проверка доступа
    if (isDayLocked(dayId)) {
        showLockedMessage();
        return;
    }
    
    // Загрузка Markdown
    const response = await fetch(`content/${dayId}.md`);
    const markdown = await response.text();
    
    // Рендеринг
    const html = marked.parse(markdown);
    document.getElementById('day-content').innerHTML = html;
    
    // Обработка чек-листов
    processChecklists(dayId);
}
```

#### **js/access.js** — Контроль доступа
```javascript
// Конфигурация дней
const DAYS_CONFIG = {
    day0: { title: 'Подготовка', duration: '90 мин', order: 0 },
    day1: { title: 'Vibe Coding', duration: '60 мин', order: 1 },
    day2: { title: 'Архитектура TMA', duration: '60 мин', order: 2 },
    day3: { title: 'Корзина и формы', duration: '90 мин', order: 3 },
    day4: { title: 'Интеграция с Telegram', duration: '75 мин', order: 4 },
    day5: { title: 'Платежи и API', duration: '60 мин', order: 5 },
    day6: { title: 'Автоматизация', duration: '75 мин', order: 6 },
    day7: { title: 'Аналитика и финал', duration: '60 мин', order: 7 }
};

// Коды разблокировки
const UNLOCK_CODES = {
    'DAY0': 'day0',
    'DAY1': 'day1',
    'DAY2': 'day2',
    'DAY3': 'day3',
    'DAY4': 'day4',
    'DAY5': 'day5',
    'DAY6': 'day6',
    'DAY7': 'day7'
};

// Получить список разблокированных дней
function getUnlockedDays() {
    const unlocked = localStorage.getItem('unlocked_days');
    return unlocked ? JSON.parse(unlocked) : [];
}

// Сохранить разблокированные дни
function saveUnlockedDays(days) {
    localStorage.setItem('unlocked_days', JSON.stringify(days));
}

// Проверка: заблокирован ли день
function isDayLocked(dayId) {
    const unlocked = getUnlockedDays();
    return !unlocked.includes(dayId);
}

// Разблокировка дня кодом
function unlockDay(code) {
    const dayId = UNLOCK_CODES[code.toUpperCase()];
    
    if (!dayId) {
        return { success: false, message: 'Неверный код' };
    }
    
    const unlocked = getUnlockedDays();
    
    if (unlocked.includes(dayId)) {
        return { success: true, message: 'День уже разблокирован', alreadyUnlocked: true };
    }
    
    unlocked.push(dayId);
    saveUnlockedDays(unlocked);
    
    return {
        success: true,
        message: `День "${DAYS_CONFIG[dayId].title}" разблокирован!`,
        dayTitle: DAYS_CONFIG[dayId].title
    };
}

// Разблокировка из URL (когда бот передаёт параметр)
function checkUnlockFromURL() {
    const urlParams = new URLSearchParams(window.location.search);
    const unlockCode = urlParams.get('unlock');
    
    if (unlockCode) {
        const result = unlockDay(unlockCode);
        if (result.success && !result.alreadyUnlocked) {
            showToast(`✅ ${result.message}`);
        }
    }
}

// Вызвать при загрузке страницы
window.addEventListener('DOMContentLoaded', checkUnlockFromURL);
```

#### **js/progress.js** — Управление прогрессом
```javascript
// Получить прогресс по всем дням
function getProgress() {
    const progress = localStorage.getItem('course_progress');
    return progress ? JSON.parse(progress) : initializeProgress();
}

// Инициализация структуры прогресса
function initializeProgress() {
    const progress = {};
    Object.keys(DAYS_CONFIG).forEach(dayId => {
        progress[dayId] = {
            completed: false,
            checklists: [] // Массив булевых значений
        };
    });
    saveProgress(progress);
    return progress;
}

// Сохранить прогресс
function saveProgress(progress) {
    localStorage.setItem('course_progress', JSON.stringify(progress));
}

// Отметить день как пройденный
function markDayCompleted(dayId) {
    const progress = getProgress();
    progress[dayId].completed = true;
    saveProgress(progress);
    updateProgressBar();
}

// Обновить чек-лист
function updateChecklist(dayId, checklistIndex, checked) {
    const progress = getProgress();
    progress[dayId].checklists[checklistIndex] = checked;
    saveProgress(progress);
}

// Получить общий прогресс
function getTotalProgress() {
    const progress = getProgress();
    const completed = Object.values(progress).filter(d => d.completed).length;
    const total = Object.keys(DAYS_CONFIG).length;
    return { completed, total, percentage: Math.round((completed / total) * 100) };
}

// Обновить прогресс-бар на главной
function updateProgressBar() {
    const { completed, total, percentage } = getTotalProgress();
    document.getElementById('progress-text').textContent = `Прогресс: ${completed}/${total}`;
    document.getElementById('progress-fill').style.width = `${percentage}%`;
}

// Обработка чек-листов в Markdown контенте
function processChecklists(dayId) {
    const checkboxes = document.querySelectorAll('input[type="checkbox"]');
    const progress = getProgress();
    
    checkboxes.forEach((checkbox, index) => {
        // Восстановить состояние из прогресса
        if (progress[dayId].checklists[index]) {
            checkbox.checked = true;
        }
        
        // Обработчик изменения
        checkbox.addEventListener('change', (e) => {
            updateChecklist(dayId, index, e.target.checked);
            vibrate('light');
        });
    });
}
```

#### **js/app.js** — Основная логика главной страницы
```javascript
// Рендеринг списка дней
function renderDays() {
    const container = document.getElementById('days-container');
    const unlocked = getUnlockedDays();
    
    Object.entries(DAYS_CONFIG).forEach(([dayId, dayInfo]) => {
        const isLocked = !unlocked.includes(dayId);
        const progress = getProgress();
        const isCompleted = progress[dayId]?.completed || false;
        
        const card = document.createElement('div');
        card.className = `day-card ${isLocked ? 'locked' : 'unlocked'} ${isCompleted ? 'completed' : ''}`;
        
        if (isLocked) {
            card.innerHTML = `
                🔒
                
                    ${dayInfo.title}
                    Введите код от преподавателя
                    
                        Ввести код
                    
                
            `;
        } else {
            card.innerHTML = `
                ${isCompleted ? '✅' : '📖'}
                
                    День ${dayInfo.order}: ${dayInfo.title}
                    ${dayInfo.duration}
                
                Открыть
            `;
        }
        
        container.appendChild(card);
    });
    
    updateProgressBar();
}

// Промпт для ввода кода
function promptUnlockCode(dayId) {
    const code = prompt('Введите код разблокировки:');
    if (code) {
        const result = unlockDay(code);
        if (result.success) {
            showToast(`✅ ${result.message}`);
            renderDays(); // Перерисовать список
        } else {
            showToast(`❌ ${result.message}`);
        }
    }
}
```

## BACKEND (TELEGRAM BOT)

### 1. Структура бота

**Основной файл (bot/src/index.js):**
```javascript
import { Bot, InlineKeyboard } from 'grammy';
import dotenv from 'dotenv';
import { UNLOCK_CODES } from './config/codes.js';

dotenv.config();

const bot = new Bot(process.env.BOT_TOKEN);
const TMA_URL = process.env.TMA_URL || 'https://your-course.vercel.app';

// Команда /start
bot.command('start', async (ctx) => {
    const keyboard = new InlineKeyboard()
        .webApp('📚 Открыть курс', TMA_URL);
    
    await ctx.reply(
        '👋 Добро пожаловать на курс "Как создавать цифровые проекты с помощью ИИ"!\n\n' +
        '📱 Нажмите кнопку ниже, чтобы начать обучение.\n\n' +
        '💡 Ваш прогресс сохраняется автоматически.\n\n' +
        '🔓 Чтобы разблокировать новый день, используйте команду /unlock и введите код, который дал преподаватель.',
        { reply_markup: keyboard }
    );
});

// Команда /help
bot.command('help', async (ctx) => {
    await ctx.reply(
        '📖 Как пользоваться курсом:\n\n' +
        '1. Нажмите "Открыть курс"\n' +
        '2. Выберите день для изучения\n' +
        '3. Читайте материалы и отмечайте чек-листы\n' +
        '4. После встречи с преподавателем получите код\n' +
        '5. Используйте /unlock чтобы разблокировать день\n\n' +
        '❓ Вопросы? Напишите @your_username'
    );
});

// Команда /unlock
bot.command('unlock', async (ctx) => {
    await ctx.reply(
        '🔓 Введите код разблокировки, который дал преподаватель:\n\n' +
        'Например: DAY1\n\n' +
        '💡 Код можно получить после прохождения встречи.'
    );
    
    // Устанавливаем флаг ожидания кода
    ctx.session = { awaitingCode: true };
});

// Обработка текстовых сообщений (коды)
bot.on('message:text', async (ctx) => {
    const code = ctx.message.text.toUpperCase().trim();
    
    // Проверка кода
    if (UNLOCK_CODES[code]) {
        const dayId = UNLOCK_CODES[code];
        
        // Формируем URL с параметром unlock
        const unlockURL = `${TMA_URL}?unlock=${code}`;
        
        const keyboard = new InlineKeyboard()
            .webApp('📚 Открыть курс', unlockURL);
        
        await ctx.reply(
            `✅ Код принят!\n\n` +
            `День разблокирован. Откройте курс чтобы увидеть новые материалы.`,
            { reply_markup: keyboard }
        );
    } else {
        await ctx.reply(
            '❌ Неверный код.\n\n' +
            'Проверьте код у преподавателя или используйте /unlock чтобы попробовать снова.'
        );
    }
});

// Запуск бота
bot.start();
console.log('✅ Бот запущен');
```

## КОНТРОЛЬ ДОСТУПА

### Схема работы
```
1. Преподаватель проводит встречу
   ↓
2. В конце встречи даёт код: "DAY2"
   ↓
3. Ученик открывает бота
   ↓
4. Вводит /unlock
   ↓
5. Вводит код "DAY2"
   ↓
6. Бот проверяет код в UNLOCK_CODES
   ↓
7. Если код верный → отправляет кнопку с URL: TMA_URL?unlock=DAY2
   ↓
8. Ученик кликает кнопку
   ↓
9. TMA открывается, JavaScript считывает параметр ?unlock=DAY2
   ↓
10. Функция unlockDay('DAY2') разблокирует день
    ↓
11. День сохраняется в LocalStorage
    ↓
12. Ученик видит открытый день
```

### Безопасность

**Уровень защиты:** Базовый (достаточный для бесплатного образовательного курса)

**Потенциальные уязвимости:**
- Ученики могут передавать коды друг другу
- LocalStorage можно очистить
- Коды простые и предсказуемые

**Меры защиты (опционально для v2):**
- Уникальные коды для каждого ученика
- Серверная проверка доступа (Supabase)
- Ограничение по времени действия кодов
- Привязка к Telegram ID

**Для текущей задачи:** Базовой защиты достаточно.
